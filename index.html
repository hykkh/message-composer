Groq API 키 발급 (완전 무료, 결제 없음)

1. console.groq.com/keys 접속
2. Google 계정으로 간편 가입
3. "Create API Key" 클릭
4. 생성된 키(gsk_...) 복사 → 위에 붙여넣기

결제 등록 없이 완전 무료!
Gemini API 키 발급

1. aistudio.google.com/apikey 접속
2. Google 계정 로그인
3. "API 키 만들기" 클릭
4. 생성된 키 복사 → 위에 붙여넣기
Claude API 키 발급

1. console.anthropic.com/settings/keys 접속
2. Anthropic 계정 로그인
3. "Create Key" 클릭
4. 생성된 키(sk-ant-...) 복사 → 위에 붙여넣기
키는 이 기기에만 저장됩니다.
